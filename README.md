# CSD 380 – DevOps Group Assignment 10.2

This project is part of the **CSD 380 DevOps course** and demonstrates the use of a **shared Git repository workflow**, including branching, pull requests, and collaboration between developers.

## Project Purpose

The goal of this assignment is to practice:

- Working in a shared Git repository
- Creating and managing branches
- Updating code collaboratively
- Submitting pull requests and merging changes
